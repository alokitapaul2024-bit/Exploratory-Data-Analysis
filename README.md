# 📊 Exploratory Data Analysis — Titanic Dataset

## 🛠️ What I Built
-An exploratory data analysis project in Python using the real-world Titanic dataset of 891 passenger records. The analysis uses visualizations and statistics to uncover what factors influenced passenger survival.
-It was made using a Jupyter Notebook which plays a crucial role in Data science or Data analysis fields

## 🚩 Problem Statement
Given passenger data, can we identify patterns that determined who survived and who didn't? Questions like:
- Did gender affect survival chances?
- Did passenger class or fare determine survival?
- How did age play a role?

## 🔍 My Approach
1. **Data Cleaning** — Handled missing values in Age and Cabin, dropped irrelevant columns
2. **Univariate Analysis** — Histograms and boxplots to understand individual variables
3. **Bivariate Analysis** — Countplots and scatterplots to find relationships between variables
4. **Correlation Heatmap** — Identified strongest predictors of survival
5. **Visualization using pairplot** — To Summarize the visualization area

## 🔐 Key Findings
- **61% died, 39% survived** — dataset is imbalanced toward non-survivors
- **Gender was the strongest factor** — female survival rate ~74% vs male ~19%
- **Class mattered significantly** — 1st class survived most, 3rd class had the most deaths
- **Fare correlated with survival** — higher paying passengers survived more
- **Age had weak impact alone** — children slightly favored, but class/fare dominated
